## üí° About The Project

**Resume Builder** is a powerful, lightweight, and **privacy-focused** web application designed to help students and professionals create stunning resumes. 

Unlike other tools that force you to create accounts or pay for downloads, this project runs **100% in your browser**. Your data never leaves your device.

## üöÄ Getting Started

You can run this project locally on your machine without installing any complex dependencies.

### Prerequisites
* A modern web browser (Chrome, Firefox, Edge, Safari).

### Installation

1.  **Clone the repository**
    ```bash
    git clone [https://github.com/shubhambelbase/Resume_builder.git](https://github.com/shubhambelbase/Resume_builder.git)
    ```

2.  **Navigate to the project directory**
    ```bash
    cd Resume_builder
    ```

3.  **Launch the App**
    * Simply double-click the `index.html` file.
    * *Or* use a simple HTTP server (e.g., Live Server in VS Code) for the best experience.

## üìñ How to Use

1.  **Fill Details:** Use the left-side editor to enter your Personal Info, Education, and Experience.
2.  **Pick a Style:** Use the dropdown menu to switch between **5 different templates**.
3.  **Color It:** Use the color picker to set a custom theme color (e.g., `#FF5733`).
4.  **Add a Photo:** Upload your profile picture; the app automatically optimizes it.
5.  **Download:** Click the **"Save as PDF"** button to download your print-ready resume.
